一、远程链接

  • 服务器Linux并且电脑是Windows

    • 远程链接工具: Xshell和SecureCRT

  • 服务器Window Server并且电脑Windows

    • 远程链接工具: 远程桌面链接

远程链接服务器

下载Xshell并安装。打开软件会弹出会话框，选择新建会话，然后进行相关配置的设置

安装Node.js

  • 使用wget命令下载安装Node.js

该安装包是编译好的文件，解压之后，在bin文件夹中就已存在node和npm

wget https://nodejs.org/dist/v6.10.2/node-v6.10.2-linux-x64.tar.xz
  • 解压文件
tar xvf node-v6.10.2-linux-x64.tar.xz
  • 安装路径移动

软件默认安装在/root/node-v6.10.2-linux-x64/目录下。如果需要将该软件安装到其他目录（如：/opt/node/）下，请进行如下操作

mkdir -p /opt/node/
mv /root/node-v6.10.2-linux-x64/* /opt/node/
rm -f /usr/local/bin/node
rm -f /usr/local/bin/npm
ln -s /opt/node/bin/node /usr/local/bin/node
ln -s /opt/node/bin/npm /usr/local/bin/npm
  • 验证Node和npm版本
node -v 
npm -v

安装mongodb

参考: Nodejs 部署到阿里云全过程

软件安装位置：/usr/local/mongodb

数据存放位置：/var/mongodb/data

日志存放位置：/var/mongodb/logs

  • cd /usr/local路径

  • 安装mongodb

wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-3.4.5.tgz
