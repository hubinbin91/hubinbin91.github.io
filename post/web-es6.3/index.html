ES6 类

简介

和Java定义类相类似

class Point {
  // 构造函数
  constructor (x,y) {
    this.x = x
    this.y = y
  }
  // 类方法
  toString () {
    return '(' + this.x + ', ' + this.y + ')'
  }
}

注意事项：

  1. typeof Point // “function”

  2. Object.keys(Point.prototype) // []

  3. Object.getOwnPropertyNames(Point.prototype) // [“constructor”,“toString”]

  4. 一个类必须有constructor方法

  5. 类方法调用必须要使用new

  6. 不存在变量提升

  7. 类的方法内部如果含有this，它默认指向类的实例

方法提取出来单独使用，this会指向该方法运行时所在的环境

constructor() {
  this.printName = this.printName.bind(this);
