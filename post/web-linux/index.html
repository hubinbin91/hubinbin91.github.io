Ubuntu分区

/: 系统根目录

/usr: 包含一般不需要修改的应用程序，命令程序文件，程序库，手册

/var: 存放系统产生的经常变化的文件

/home: 存放用户文件的主目录，用户数据

/tmp: 存放启动时产生的临时文件

swap: 交换区。建议容量为内存的2倍
