HTML标签属性

doctype

文档类型声明，告诉标记语言解析器用何种文档类型定义DTD来解析严格版本、过渡版本以及框架版本

HTML 4.01规定了三种文档类型：Strict、Transitional 以及 Frameset。

XHTML 1.0规定了三种XML文档类型：Strict、Transitional 以及 Frameset。

lang

标签元素内容使用何种语言

display:none和visibility:hidden的区别

display:none 隐藏对应的元素，在文档布局中不再给它分配空间

visibility:hidden 隐藏对应的元素，但是在文档布局中仍保留空间
